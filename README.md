# iPatient
iPatient is a mobile app which helps patients find right doctor, book visits and without approaching physical registration window pick up their visit by scanning QR code in a medical facility. For facility staff it gives functionalities such as managing facilities info, doctors info and generating QR code for patients, who thanks to this will be able to receive theirs visit automatically.
